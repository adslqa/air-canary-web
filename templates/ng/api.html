REST API v1

The Air Canary REST API provides access to air quality data through HTTP REST requests.

Base URL

http://api.aircanary.com

Authentication

Currently no authentication is required.

Request Limit

API requests are limited to 1000 per day per IP address.

Site Resource

Request Description
GET /site Get all sites
GET /site/:code Get data for individual site

GET /site

Gets all sites

GET http://api.aircanary.com/site
